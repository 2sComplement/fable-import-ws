Fable bindings for [WS](https://github.com/websockets/ws).
=======
[![nuget](https://badge.fury.io/nu/Fable.Import.Ws.svg)](https://badge.fury.io/nu/Fable.Import.Ws)



### Using
1. `paket add Fable.Import.Ws`
2. `yarn add ws`

### Building the package
1. `dotnet restore`
2. `dotnet build`
3. `dotnet pack`
